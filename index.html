تعلم HTML و CSS و JavaScript لبناء واجهات مميزة

في هذه الدورة، ستتعلم أساسيات تصميم مواقع الويب باستخدام HTML و CSS، وكذلك كيفية إضافة التفاعلية باستخدام JavaScript. الدورة مناسبة للمبتدئين الذين يرغبون في تعلم كيفية بناء واجهات مهنية وجذابة دون الحاجة إلى خبرة سابقة في البرمجة.

أهداف الدورة

  • ✔️ تعلم أساسيات HTML 5 و SEO
  • ✔️ تصميم مواقع باستخدام CSS
  • ✔️ تعلم JavaScript لتحسين التفاعلية والمرئيات
